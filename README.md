Blog in Spring boot Example code
This project has following functionality

1. Blog Post create
2. Blog Post update
3. Blog Post delete
4. Blog Post listing with pagination
5. Blog Post comments
6. Comment edit
7. Comment delete
8. Admin Login
9. User registration
10. User Role management
